    Overview

    A GPU-accelerated, fully managed container orchestrator optimized for modern AI workloads.

    Developer Experience¶

    End User Self Service¶

    End user self-service experience to configure, launch and use a Managed Kubernetes Cluster with GPUs.

    AI/ML Ready¶

    Accelerate setup with pre-installed GPU Operator for Kubernetes, enabling quick deployments for AI and machine learning workloads.

    Management¶

    Multi-Version Support¶

    Support for multiple Kubernetes versions with automated upgrade paths and rollback capabilities, allowing users to stay current with the latest features.

    High Availability Control Plane¶

    Multi-master setup with automated failover and distributed etcd for maximum reliability, providing 99.9% uptime SLA.

    Automated Cluster Lifecycle¶

    Automated cluster provisioning, scaling, upgrades, and patching with zero downtime, enabling hands-off operations.

    Integrated Metrics¶

    View Kubernetes metrics and GPU metrics from the user console, providing real-time usage monitoring.

    Metering¶

    Precise, per-hour metering for tracking resource usage to support billing and reporting, ensuring accurate resource accounting.

    Info

    Managed Kubernetes Clusters are metered by the hour with a minimum usage of 1 hour.

    Block Storage¶

    Turnkey integration with CSIs from Rook/Ceph, DDN, Weka, Vast Data and others, providing integrated storage solutions.

    Networking¶

    CNI Plugin Support¶

    Choose from Cilium, Calico, or Flannel for pod networking and network policies, providing flexible networking options.

    Load Balancer Integration¶

    Native integration with cloud load balancers and ingress controllers for automatic load balancing.

    Security¶

    RBAC & Security Policies¶

    Built-in RBAC, Pod Security Standards, and Network Policy enforcement for enterprise security.

    Buzz Zero Trust Kubectl¶

    Enforce Zero Trust principles for kubectl access using Buzz's Access Proxy with centralized authorization, auditing, and identity-based access controlsâ€”without VPN or bastion hosts, providing secure developer access.

    Performance¶

    Latest GPUs¶

    Support for state-of-the-art GPU models such as H100, A100, and more from NVIDIA, AMD, and Intel.
